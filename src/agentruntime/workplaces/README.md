# Carvis Workplaces

Each role gets one isolated workplace:

- `workplaces/manager`
- `workplaces/writer`
- `workplaces/artist`
- `workplaces/researcher`
- `workplaces/engineer`

Each workplace uses the same file names:

- `input.md`
- `plan.md`
- `log.md`
- `result.md`

The helper functions in this directory create and update those files for local smoke tests and later PID Agent integration.
